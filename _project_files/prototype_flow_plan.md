# Glaze Prototype Plan

I will connect the existing screens into a functional prototype flow to demonstrate the "Vitreous" user experience.

### Navigation Flow:
1. **From Library ({{DATA:SCREEN:SCREEN_7}}):**
   - Click a template card (e.g., "Python Code Auditor") -> Go to **Bilingual Editor** ({{DATA:SCREEN:SCREEN_5}}).
   - Click the "+ New Template" button -> Open the **New Template** modal ({{DATA:SCREEN:SCREEN_3}}).
   - Use the Search/Collections sidebar -> Go to **Search Results** ({{DATA:SCREEN:SCREEN_6}}).

2. **From Bilingual Editor ({{DATA:SCREEN:SCREEN_5}}):**
   - Click "Save Changes" or navigation links -> Return to **Library** ({{DATA:SCREEN:SCREEN_7}}).
   - Sidebar navigation -> Switch between Library, Collections, etc.

3. **From New Template Modal ({{DATA:SCREEN:SCREEN_3}}):**
   - Click "Create Template" -> Go to **Bilingual Editor** ({{DATA:SCREEN:SCREEN_5}}).
   - Click "Cancel" or Close -> Return to **Library** ({{DATA:SCREEN:SCREEN_7}}).

4. **From Search Results ({{DATA:SCREEN:SCREEN_6}}):**
   - Click a result card -> Go to **Bilingual Editor** ({{DATA:SCREEN:SCREEN_5}}).
   - Navigation links -> Return to **Library** ({{DATA:SCREEN:SCREEN_7}}).
